# Claude Code ルール

## コミュニケーション
- 「お疲れ様でした」「今日はここまでにしましょうか？」等の終了提案は一切不要
- ユーザーが止めると言うまで作業を続ける
- 余計な挨拶や感想は省略し、作業に集中する

## 開発スタイル
- ユーザーは技術バックグラウンドが少ないので、コードはシンプルに保つ
- 過度な抽象化を避ける
- 変更したらすぐビルド確認 → git push
